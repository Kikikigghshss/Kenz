Tentang Saya

Halo! Saya adalah Kenzo, seorang siswa SMK yang saat ini sedang belajar dasar-dasar pengembangan web. Saya sangat antusias untuk memulai perjalanan saya dalam HTML dan CSS dan berharap dapat mengaplikasikan apa yang saya pelajari dalam proyek-proyek nyata.

Meskipun saya baru memulai, saya bersemangat untuk mengembangkan keterampilan saya dan membangun dasar yang kuat dalam teknologi web. Tujuan saya adalah untuk meningkatkan kemampuan saya dan menciptakan situs web yang menarik dan fungsional.

Selain pengembangan web, saya juga memiliki minat dalam desain digital dan fotografi, yang melengkapi pengalaman belajar saya dan memberikan saluran kreatif. Saya bersemangat untuk menjelajahi teknologi baru dan terus memperbaiki keterampilan saya melalui proyek-proyek praktis.

Terima kasih telah mengunjungi portofolio saya. Semoga Anda menikmati melihat karya-karya saya!
